TED talks:10 top time-saving tech tips

Published: May 16 2013

今天看到David Pogue在TED关于节省时间技巧的演讲，转述到这里，顺便加上我自己平时使用的技巧。

web

  • space => scroll down one page
  • shift + space => scroll up one page
  • ctrl + + => 放大显示文本
  • ctrl + - => 缩小显示文本

more than that

you can use these with any web browser, if you use ‘vimperater’ or ‘vimium’, you can throw away mouse when surfing the internet. Or you can use shortcuts listed below if you don’t want to learn to use above extensions.

  • ctrl + f(ind)
  • ctrl + p(rint) => print current page
  • ctrl + j => show download page
  • ctrl + t => open a new tab
  • ctrl + w => exit current page

presentation

  • b => 黑屏
  • w => 白屏

more than that

以上适用于微软powerpoint, keynote等。在做演示的时候，让投影暂时黑屏或白屏，观众将注意力集中于演讲人，而不是幻灯片，要重新显示内容只需再按一次即可。

  • F5 => begin display from the start
  • shift + F5 => begin display from current slide

search

  • define:keyword 在google搜索框这样输入，即可查词。
  • 单位转换 10 inches in cm 即可得到10英尺对应的厘米数; 10usd in rmb 即可得到10美刀对应的人民币。

more on search

  • 还可以把google当计算器用：235 * 45 / (34-3), 4 ** 5(4的5次方)
  • site:njau.edu.cn keyword #search in njau.edu.cn for something,
    you may also use site:.cn keyword to search in ‘all domain under cn’ for something
  • filetype:pdf some name of books #search for a book which format is pdf
